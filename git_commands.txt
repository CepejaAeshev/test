git init        - создаёт отслеживание в этой папке
git add file    - если создали новый файл то нужно его добавить в отслеживание  | можно добавлять файлы вручную нажимая
git add .       - добавляет в отслеживание все файлы в папке                    | на + во вкладке Source Control
git status      - все отслеживаемые файлы
git commit -m "текст коммита"   - сделать коммит в выбраной ветке               | в коммит идут только отслеживаемые файлы
git log             - посмотреть логи комитов в ветке
git checkout хеш    - вернуться к комиту по хешу
git checkout название_ветки - вернуться к последнему комиту в указанной ветке   | можно переключиться через кнопку справа внизу
git branch название_ветки   - создать ветку начиная с комита на котором мы находимся    
git merge название_ветки    - слить указанную ветку с той, в которой мы находимся
git remote add куда_загрузить что_загрузить                                     | git remote add origin main
git clone ссылка_на_репозиторий - скачать репозиторий                           | при этом терминал должен быть открыт в нужной папке
